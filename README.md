# 🚗 Accident Risk Predictor

This project predicts accident risk using machine learning and sends SMS alerts using Twilio.

## 🔥 Features
- Predicts accident risk based on speed, weather, and time
- User-friendly UI using Streamlit
- Sends SMS alert using Twilio

## 🛠️ Technologies Used
- Python
- Streamlit
- Machine Learning (Random Forest)
- Twilio API

## 📊 Dataset
- Small sample dataset used for prediction

## ▶️ How to Run

```bash
pip install streamlit
streamlit run app.py
